For users who are new to build servers and would like some background information on what they are doing for learning please go here

STEP #1 - Get the Client!

Get the Client: >>> DOWNLOAD HERE <<< OR >>> HERE <<<

STEP #2 - Configure the Client!

Configure your Client Get this ZIP package for Crucible and follow the instructions in the Readme.md file to "complete" your client configuration.
NOTE: Do NOT run the Client from the icon [the launcher] you get after installation. Change it to run the /bin/exefile.exe instead of the eve.exe

STEP #3 - Get the Server!

TWO Options:

  1. USE PRE-BUILT BINARIES: pre-compiled binaries for your platform) - Recommended for New Users You will need both *.7z files and the binary.
  2. BUILD IT - Build Instructions can be found below in the LEFT column (Crucible is our Current Development)

STEP #4 - Configure and Setup the Server!

Once you either built or downloaded pre-built server, follow these instructions to configure and setup your server.

STEP #5 - Test and Report!

Yes! We know there are bugs and missing features!
Click here to see a list of known working and missing/broken features of the EVE Online experience.
You can request to be part of the Testers Group. Just head into our IRC channel and get to know some of them, request a forum account and start testing against the Project Status.
See this article for specific information on how to go about testing and creating reports and submitting bug/feature reports on our Github project page. If you want to report bugs report them here
