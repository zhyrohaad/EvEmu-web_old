About:

Thank you for playing EVE on AlphaTest Server, please note this server is intent for test fly Capital ships and many features are not implemented for full experience. Server is expected to have random crash, lag and blackscreen. Admin is working on to provide latest updates to server stability and feature addons.

Game Play Info:
* When first login close Aura Tutorial to prevent later black screen.
* On Chatbox, type /summon [ID] or copy paste ID to summon items. Minimize it than close it.
* Activiate ships that have no graphic may cause blackscreen or server crash when undock.
* If you believe your account is corrupted please create a new one.
* Module mount and Learn Skills are not supported at this time.
