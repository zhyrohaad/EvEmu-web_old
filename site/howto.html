How to Guide:

1. Install EVE-Online Client to a dir.
2. Launch client but do not download updates and close client.
3. Go and open EVE/Cache/pref.ini, find and edit:

advanced=1
debug=1
network=1
networkAdvanced=1
port=26001

Note: Some of these don't exist, simply add them in.

4. Go to EVE/stdlib/urllib2.py, use notepad to open, Find "def urlopen" and replace the body and save:

def urlopen(url, data=None):
global _opener
rewrites = {
'http://www.eve-online.com' : 'http://60.240.37.49:80',
'http://myeve.eve-online.com' : 'http://60.240.37.49:80/myeve',
'https://secure.eve-online.com' : 'http://60.240.37.49:80/secure',
'http://support.eve-online.com' : 'http://60.240.37.49:80/supprt'
}
ourl = url.get_full_url()
matched = False
for k,v in rewrites.items():
if ourl[:len(k)] == k:
newurl = v+ourl[len(k):]
url = Request(newurl, url.get_data(), url.get_headers())
matched = True
if not matched:
url = None #prevent it from going anywhere we are not in control of
if _opener is None:
_opener = build_opener()
return _opener.open(url, data)

5. Launch EVE and on server box click on 'Transquility' text to edit, replace with "60.240.37.49" and connect with port "26001".

IMPORTANT: Currently the game tutorial "Aura" may causes problems, so always permanently disable the tutorial as soon as you get logged in.
